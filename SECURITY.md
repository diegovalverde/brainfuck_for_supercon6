# Security Policy

## Supported Version

This repository is maintained as a demo/experiment project. Security fixes are
applied on the `main` branch.

## Reporting a Vulnerability

Please report security issues privately using GitHub Security Advisories:

- Go to the repository Security tab.
- Create a private vulnerability report.

Do not open public issues for vulnerabilities.

## Notes

- This project includes vendored runtime assets under `js/lib/pyodide`.
- Treat this repository as non-production software unless you perform your own
  dependency and hosting hardening.
